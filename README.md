# ReactNative
 Workshop for mobile application developing using React native
 
 ## Start the emulation
 
 ```
 expo init <app-name>
```

```
cd <appfolder>
```
```
npm start
```

 
 ### Workshop 1
 Highlights
* Create components
* States
* Props
* Styling

 ### Workshop 2
 Highlights
* Extra styling
* Customisable component
* using props for styling and passing data
